# MrCode

Third-party AI/ML library for the Aurora programming language.

Provides extern C++ bindings for machine learning, NLP, computer vision, vector search, agent frameworks, chat, autograd, media generation, and more.

## Install

```
voss install MrCode
```

## Usage

```aura
import MrCode

let model = MrCode::model_create("transformer")
let img   = MrCode::image_gen("a scenic mountain landscape")
let text  = MrCode::generate("Once upon a time")
```
